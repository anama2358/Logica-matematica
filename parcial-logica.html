🔵 Definición de Conjunto

Un conjunto es una colección bien definida de objetos distintos, llamados elementos o miembros del conjunto. Los elementos pueden ser números, letras, personas, objetos físicos, o incluso otros conjuntos.

Características principales:
  • Los elementos están bien definidos (no hay ambigüedad)
  • No importa el orden de los elementos
  • Los elementos no se repiten
  • Se representan generalmente con letras mayúsculas
Ejemplo Visual:

Imagina una caja que contiene diferentes frutas:

A = {manzana, naranja, plátano, uva}

Cada fruta es un elemento del conjunto A.

📝 Notación de Conjuntos

Los conjuntos se representan usando símbolos específicos que nos permiten expresar relaciones y operaciones de manera precisa.

Símbolos Fundamentales:
  • { } - Llaves para delimitar el conjunto
  • ∈ - "Pertenece a" (a ∈ A significa que a es elemento de A)
  • ∉ - "No pertenece a" (b ∉ A significa que b no es elemento de A)
  • ∅ o { } - Conjunto vacío
  • ⊆ - "Subconjunto de"
  • ⊂ - "Subconjunto propio de"
  • |A| - Cardinalidad (número de elementos)
📌 Ejemplos de Notación en Conjuntos
  • Notación de conjunto por extensión o enumerativa: (Conjunto de vocales) V={a,e,i,o,u}
  • Notación de conjunto por comprensión o descriptiva: B={x∈N∣x <10} "B es el conjunto de todos los números naturales menores que 10" .
  • Notación de pertenencia y no pertenencia: x∈A significa: "x pertenece a A".
  • Notación de subconjuntos: A⊆B: A es subconjunto de B (puede ser igual o menor).

📋 Descripción de Conjuntos

Por Extensión (Enumeración)

Se listan todos los elementos del conjunto separados por comas y encerrados entre llaves.

Ejemplos por Extensión:
A={2,4,6,8,10} - Se lee: "A es el conjunto de los números 2, 4, 6, 8 y 10."
B = {a, e, i, o, u}
C = {rojo, azul, verde, amarillo}
D = {lunes, martes, miércoles, jueves, viernes, sábado, domingo}

Por Comprensión

Se describe una propiedad que caracteriza a todos los elementos del conjunto.

Ejemplos por Comprensión:
A = {x ∈ N∣ x es par y x ≤ 10} - Se lee: "A es el conjunto de todos los números naturales que son pares y menores o iguales a 10."
B = {x | x es una vocal del alfabeto español}
C = {x | x es un color primario o secundario}
D = {x | x es un día de la semana}
E = {x ∈ ℕ | x² < 25}={1, 2, 3, 4}

🎯 Diagramas de Venn

Los diagramas de Venn son representaciones gráficas que muestran las relaciones entre conjuntos mediante círculos u otras formas geométricas.

Usos y Utilidades

  • Visualizar relaciones entre conjuntos
  • Resolver problemas de lógica y probabilidad
  • Organizar información de manera clara
  • Facilitar el aprendizaje de operaciones entre conjuntos
  • Análisis de datos y clasificación
Ejemplo Práctico:

Estudiantes en una clase:

  • A = Estudiantes que practican fútbol
  • B = Estudiantes que practican baloncesto
  • A ∩ B = Estudiantes que practican ambos deportes
  • A ∪ B = Estudiantes que practican al menos un deporte

🔢 Tipos de Conjuntos

Finito

|A| = n

Número limitado de elementos

A = {1, 2, 3, 4}

Infinito

|A| = ∞

Número ilimitado de elementos

ℕ = {1, 2, 3, ...}

Vacío

|∅| = 0

Sin elementos

∅ = { }

Unitario

|A| = 1

Exactamente un elemento

A = {5}

🔗 Relaciones entre Conjuntos

Equivalentes

A ~ B

Misma cardinalidad

{a, b, c} ~ {1, 2, 3}

Disjuntos

A ∩ B = ∅

Sin elementos comunes

{1, 2} ∩ {3, 4} = ∅

Subconjunto

A ⊆ B

Todos los elementos de A están en B

{1, 2} ⊆ {1, 2, 3}

🎯 Conjuntos Especiales

Universal (U)

Referencia

Contiene todos los elementos del contexto

U = {1, 2, 3, 4, 5, 6, 7, 8, 9}

Complementario

A' = U - A

Elementos que NO están en A

Si A = {1, 3, 5}, entonces A' = {2, 4, 6, 7, 8, 9}
Clasificación Completa de Tipos:

📊 Por Cardinalidad

  • Finito: A = {a, b, c, d} - Tiene exactamente 4 elementos
  • Infinito: ℝ = {números reales} - Tiene infinitos elementos
  • Vacío: A = {x | x es par e impar} = ∅
  • Unitario: A = {5} - Tiene exactamente un elemento

🔗 Por Relaciones

  • Equivalentes: A = {1, 2, 3} ~ B = {a, b, c}
  • Disjuntos: A = {1, 2} y B = {3, 4} → A ∩ B = ∅
  • Subconjunto: A = {1, 2} ⊆ B = {1, 2, 3, 4}

🎯 Especiales

  • Universal: U = {todos los elementos del contexto}
  • Complementario: A' = U - A (lo que NO está en A)

Operaciones entre conjuntos


🔗 Unión de Conjuntos

Definición: La unión de dos conjuntos A y B es el conjunto que contiene todos los elementos que pertenecen a A, a B, o a ambos.
Ejemplo Práctico 1:

A ∪ B = {x | x ∈ A ∨ x ∈ B}


    Conjunto A

  • {1, 2, 3, 4}

    • Conjunto B

  • {3, 4, 5, 6}

    • Union conjunto A y B

  • A ∪ B= {1, 2, 3, 4, 5, 6}
Ejemplo Práctico 2:

A ∪ B = {x | x ∈ A ∨ x ∈ B}


    Conjunto A

  • {a, b, c}

    • Conjunto B

  • {c, d, e}

    • Union conjunto A y B

  • A ∪ B= {a, b, c, d, e}

🎯 Intersección de Conjuntos

Definición: La intersección de dos conjuntos A y B es el conjunto que contiene todos los elementos que pertenecen tanto a A como a B.
Ejemplo Práctico:

A ∩ B = {x | x ∈ A ∧ x ∈ B}


    Conjunto A

  • {1, 2, 3, 4}

    • Conjunto B

  • {3, 4, 5, 6}

    • Intersección conjunto A y B

  • A ∩ B= {3, 4}
Ejemplo Práctico 2:

A ∪ B = {x | x ∈ A ∨ x ∈ B}


    Conjunto A

  • {a, b, c}

    • Conjunto B

  • {c, d, e}

    • Intersección conjunto A y B

  • A ∩ B = {c}

➖ Diferencia de Conjuntos

Definición: La diferencia de A menos B es el conjunto de elementos que pertenecen a A pero no a B.
Ejemplo Práctico:

A - B = A \ B = {x | x ∈ A ∧ x ∉ B}


    Conjunto A

  • {1, 2, 3, 4}

    • Conjunto B

  • {3, 4, 5, 6}

    • Diferencia conjunto A y B

  • A - B= {1, 2}

    • 📝 Nota importante:

    La diferencia NO es conmutativa: A - B ≠ B - A
    En este ejemplo: B - A = {5, 6}
Ejemplo Práctico 2:

    Conjunto A

  • {a, b, c}

    • Conjunto B

  • {c, d, e}

    • Diferencia conjunto A y B

  • A - B = {a, b}

🔄 Complemento de Conjuntos

Definición: El complemento de un conjunto A (respecto a un universo U) es el conjunto de todos los elementos del universo que no pertenecen a A.
Ejemplo Práctico 1:

A' = U - A = {x | x ∈ U ∧ x ∉ A}


    Universo

  • {1, 2, 3, 4, 5, 6, 7, 8}

    • Conjunto A

  • {2, 4, 6}

    • Complemento de Conjunto A

  • A'= {1, 3, 5, 7, 8}
Ejemplo Práctico 2:

    Universo

  • {a, b, c, d, e, f}.

    • Conjunto A

  • {a, b, c}

    • Complemento de Conjunto A

  • Aᶜ = {d, e, f}

⚖️ Diferencia Simétrica

Definición: La diferencia simétrica de A y B es el conjunto de elementos que pertenecen a A o a B, pero no a ambos.
Ejemplo Práctico 1:

A △ B = (A - B) ∪ (B - A) = (A ∪ B) - (A ∩ B)


    Conjunto A

  • {1, 2, 3, 4}

    • Conjunto B

  • {3, 4, 5, 6}

    • Diferencia Simétrica de A △ B

  • A △ B= {1, 2, 5, 6}

💡 Propiedades importantes:

• La diferencia simétrica ES conmutativa: A △ B = B △ A
• También se puede escribir como: A ⊕ B (XOR)
Ejemplo Práctico 2:

    Conjunto A

  • {a, b, c}

    • Conjunto B

  • {c, d, e}

    • Diferencia Simétrica de A △ B

  • A Δ B = {a, b, d, e}
